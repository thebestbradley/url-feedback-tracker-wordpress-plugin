Track and manage feedback URLs for your WordPress site.

== Description ==

Feedback URL Tracker is a powerful plugin that allows you to easily track and manage feedback URLs on your WordPress site. It provides a simple interface to create, edit, and monitor feedback links, helping you gather valuable user insights.

Features:
* Create custom feedback URLs
* Track click-through rates
* Manage multiple feedback campaigns
* Integrate with your existing forms and surveys

== Installation ==

1. Upload the `feedback-url-tracker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings from the Feedback URL Tracker menu

== Frequently Asked Questions ==

= How do I create a new feedback URL? =

Navigate to the Feedback URL Tracker menu in your WordPress admin panel and click on "Add New URL".

= Can I customize the redirect behavior? =

Yes, you can set custom redirects for each feedback URL in the URL settings.
